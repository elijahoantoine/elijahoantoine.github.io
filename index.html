Projects

Aerodynamics · Computational Personal

XFOIL Python Interface

A Python tool that automates XFOIL's interactive command-line workflow — sweeping angles of attack, running multi-Reynolds number analyses, and generating publication-quality plots. Built to remove the manual overhead of airfoil analysis and deepen understanding of panel method behavior through implementation.

Python XFOIL matplotlib numpy pywinpty Panel methods Viscous analysis

Automated AoA sweeps

Specify range and step size — internal convergence stepping at 0.05° for stability, saving only the requested data points.

Multi-Reynolds number analysis

Run multiple Re numbers in one session. Results stored per Re, with CL vs alpha and drag polar comparisons across conditions.

Pressure distribution plots

Cp vs x/c with airfoil geometry overlay, coefficient annotation, multi-AoA overlays, and experimental data import for validation.

Append and resume logic

Append new sweep results to existing polar files with auto-sorted AoA. Pause mid-sweep with Ctrl+C and save partial results.

Quick start

git clone https://github.com/elijahoantoine/xfoil-interface.git
cd xfoil-interface
pip install -r requirements.txt
python XFOIL_Interface/main.py

About

I'm Elijah Antoine, a junior studying Aerospace Engineering at Penn State University, graduating May 2027. My coursework spans aerodynamics, flight dynamics, orbital mechanics, structural analysis, and systems and controls — with the space systems track as my focus.

I build projects because working through a problem from scratch is how I actually understand it. The XFOIL tool started as a way to streamline airfoil analysis and ended up teaching me as much about panel method behavior as any lecture did. The same logic drives everything else here — if I want to understand how a system works, I build it.

My longer-term interests are in spacecraft GNC, astrodynamics, and flight software. I'm planning to apply to graduate programs in aerospace engineering focused on those areas after finishing my degree.
